# SSR(Sever Side Rendering)

SSR은 Server Side Rendring의 줄임말로, 웹 페이지를 브라우저에서 렌더링하는 대신에, 서버에서 렌더링을 한다. 브라우저가 서버의 URI로 GET 요청을 보내면, 서버는 정해진 웹 페이지 파일을 브라우저로 전송한다. 그리고 서버의 웹 페이지가 브라우저에 도착하면 완전히 렌더링이 된다.
